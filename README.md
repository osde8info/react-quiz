# ReactQuiz

ReactQuiz is a React + TypeScript quiz app scaffold built with Vite.

## Scripts

- `npm run dev` starts the development server.
- `npm run build` creates a production build.
- `npm run preview` serves the production build locally.

## Structure

- `src/data` holds quiz content.
- `src/features/quiz` contains the quiz flow.
- `src/components` contains reusable UI pieces.
- `src/hooks` contains stateful quiz logic.
- `src/styles` contains global styling.
